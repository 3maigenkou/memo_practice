# メモアプリ
sinatraを使って作成したシンプルなメモアプリです。

## 使い方
### 1.ダウンロード
```
% git clone https://github.com/3maigenkou/memo_practice.git
```
### 2.bundleのインストール
```
% bundle install
```
### 3.bundleのインストール、アプリの実行
```
% bundle install
% bundle exec ruby app.rb
```
### 4.ブラウザでアクセス
[http://localhost:4567](http://localhost:4567)
